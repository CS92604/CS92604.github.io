Overview

Why this exists

The IS300 / 2JZ-GE community has a lot of scattered information around factory ECU tuning. This project is meant to turn that into one clear, tested, public workflow.

1

Identify the ECU

Document the ECU part number, Denso number, software ID, CAL IDs, and supported file family so owners know exactly what they are working with.

2

Verify the tools

Confirm the correct BitBox and BitEdit modules, what each tool does, and where the read/write, editing, and checksum responsibilities are handled.

3

Publish the workflow

Build a practical guide with screenshots, notes, limitations, safe calibration boundaries, and direct links to the required tools.

Documentation Plan

Planned workflow coverage

This page will become the public home for the verified method once testing is complete.

Core guide sections

  • Confirm ECU part number and software ID.
  • Confirm BitBox Toyota Denso Gen1 CAN / FID 24 applicability.
  • Confirm BitEdit Toyota Denso Petrol 3 editing support.
  • Read or obtain the original stock file.
  • Document checksum and write workflow requirements.
  • Review available maps and calibration limits.
  • Show what the factory ECU can and cannot realistically support.

Video / media coverage

  • ECU identification walkthrough.
  • Tool setup overview.
  • Read/write workflow notes.
  • BitEdit project/file overview.
  • Safe calibration workflow explanation.
  • Results, limitations, and next steps.
  • Long-term updates as the method is refined.
This project is focused on legitimate calibration, safe ECU workflow documentation, fuel/timing control, software identification, and proper tool usage. It is not intended for emissions defeat, illegal use, or unsafe calibration practices.
