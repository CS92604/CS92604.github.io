Overview

Why this exists

The Toyota and Lexus 2JZ-GE VVT-i community has a lot of scattered information around factory ECU tuning. This project is meant to turn that into one clear, tested, public workflow.

1

Identify the ECU

Document the ECU part number, Denso number, software ID, CAL IDs, connector requirements, and supported file family so owners know exactly what they are working with.

2

Verify software access

Confirm what software can access the ECU, what the workflow can actually do, and where reading, writing, editing, and file safety checks are handled.

3

Publish the workflow

Build a practical guide with screenshots, notes, limitations, safe calibration boundaries, and clear requirements for anyone trying to follow the same path.

Platform Coverage

Planned Toyota / Lexus coverage

This workflow is being developed around OBD2-equipped Toyota and Lexus platforms using related factory ECU architecture. The primary focus is USDM cars, especially the Lexus IS300, because that platform has the largest visible demand and the clearest test path.

Potentially relevant platforms

Toyota Altezza AS 300 / Lexus IS 300
Toyota Aristo / Lexus GS 300
Toyota Crown / Crown Majesta
Toyota Mark II / Chaser / Cresta
Toyota Origin
Toyota Progres
Toyota Soarer / Lexus SC 300
Toyota Supra
This workflow only applies to vehicles equipped with an OBD2 port. JDM models have not been tested yet. Current development is focused primarily on USDM vehicles, with the Lexus IS300 platform as the main priority due to popularity and available test data.
Documentation Plan

Planned workflow coverage

This page will become the public home for the verified method once testing is complete.

Core guide sections

  • Confirm ECU part number, software ID, and CAL IDs.
  • Confirm vehicle eligibility and OBD2 access requirements.
  • Confirm which software workflow can access the ECU.
  • Read or obtain the original stock file when possible.
  • Document safe file handling and write workflow requirements.
  • Review available calibration areas and practical limits.
  • Show what the factory ECU can and cannot realistically support.

Video / media coverage

  • ECU identification walkthrough.
  • Vehicle compatibility notes.
  • OBD2 access and setup overview.
  • Read/write workflow notes.
  • Factory file and calibration review.
  • Safe calibration workflow explanation.
  • Results, limitations, and next steps.
This project is focused on legitimate calibration, safe ECU workflow documentation, fuel/timing control, software identification, and proper tool usage. It is not intended for emissions defeat, illegal use, or unsafe calibration practices.
